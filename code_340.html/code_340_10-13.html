연습문제 페이지 340 문제 10-13 [10번] (2)-{} [11번] (1) 10 + 20은 덧셈 연산으로 결과는 30, 문자열 '!!!!'와 결합되면 숫자 30이 문자열로 변한되고 30!!!!이라는 문자열이 되므로 출력 결과는 30!!!!이 된다. (2) 처음 alert(++number)는 number를 10에서 11로 증가시키고 11을 출력한다. 두 번째 alert(++number)는 number를 11에서 12로 증가시키고 12을 출력한다. 세 번째 alert(++number)는 number를 12에서 13으로 증가시키고 13을 출력한다. (3) let test로 선언된 변수는 호이스팅 시점에서 초기화되지 않은 상태로 남아있기 때문에 test는 변수로 동작한다. 따라서 test는 함수가 아니라 변수로 간주되며 alert(test)는 test 변수에 저장된 함수 표현식 자체를 출력한다. 그러므로 출력된 결과는 다음과 같다. 함수의 소스 코드: "function () { return 10 }" [12번] =는 할당 연산자로, 값을 변수에 할당할 때 사용하고 ==는 동등 연산자로, 두 값을 비교하여 값이 같은지 확인하는 역할을 한다.
